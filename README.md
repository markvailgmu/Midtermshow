# Midtermshow

PowerShell: use **`.\build.ps1`** (not `./build.bat`) so after a good build this folder is on **PATH for that window** and you can type **`mydir`** without `.\`. Or run **`.\tools.ps1`** once per session. **`add_project_to_path.bat`** adds the folder to user PATH permanently. **`install_msvc.bat`** (admin) installs MSVC.
